# 🩸 Anemia Prediction Using Machine Learning

A machine learning project that predicts whether a person is anemic based on blood sample pixel color values and hemoglobin (Hb) levels.

---

## 📌 About the Project

Anemia is a condition where the blood lacks enough healthy red blood cells. This project uses image-derived RGB pixel data and hemoglobin values to classify patients as **Anemic** or **Non-Anemic** using multiple machine learning algorithms.

---

## 🛠️ Technologies Used

- Python 3
- Jupyter Notebook
- pandas, numpy
- scikit-learn
- seaborn, matplotlib

---

## 🤖 ML Models Used

 Models

 K-Nearest Neighbors (KNN) 
 Random Forest  
 Logistic Regression 
 Support Vector Machine (SVM)

---

## 📊 Evaluation Metrics

Each model is evaluated using:
- ✅ Accuracy Score
- 📋 Classification Report (Precision, Recall, F1-Score)
- 📊 Confusion Matrix (visualized using Yellowbrick)

---

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/anemia-prediction.git
   cd anemia-prediction
   ```

2. Install dependencies:
   ```bash
   pip install pandas numpy scikit-learn seaborn matplotlib yellowbrick
   ```

3. Open the notebook:
   ```bash
   jupyter notebook Anemia.ipynb
   ```

4. Make sure `AnemiaDataset.csv` is in the same folder as the notebook.
